Steps to Configure the Database:

Pre-reqs:
Must have xampp, nodejs installed.

Steps:
Create a Database in phpmyadmin with necessary tables. Check the Database Schema.
in my case: (accounts, balance, loans, transactions)
Modify queries in the server.js file.
Start the nodemon server by running "npm run dev" in the terminal of server.js
Start the react app with "npm run dev" in any terminal in the react app.

Admin login is admin admin.
Enjoy :)
